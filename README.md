# 🚜 Bulldozer Price Prediction (End-to-End Machine Learning Project)

> A complete machine learning project that predicts the sale price of bulldozers using historical auction data.

---

## 📌 Overview

This project focuses on building an **end-to-end regression model** to predict bulldozer prices based on various features like usage, model type, and sale conditions.

It demonstrates the full machine learning lifecycle:

* Data preprocessing
* Feature engineering
* Model training
* Evaluation
* Optimization

---

## 🎯 Problem Statement

> Given historical auction data of bulldozers, can we predict their future sale price accurately?

---

## 📊 Dataset

* Source: Kaggle ([Bluebook for Bulldozers dataset](https://www.kaggle.com/competitions/bluebook-for-bulldozers/data))
* Contains:

  * Equipment type
  * Usage hours
  * Year of manufacture
  * Sale date
  * Product size and category

---

## ✨ Features

* ✅ Data cleaning and preprocessing
* ✅ Handling missing values
* ✅ Feature engineering (date-based features, categorical encoding)
* ✅ Model training with regression algorithms
* ✅ Model evaluation using error metrics
* ✅ Hyperparameter tuning

---

## 🧠 Machine Learning Approach

* Problem Type: **Regression**
* Model Used:

  * Random Forest Regressor 🌲

---

## 📈 Evaluation Metrics

* RMSLE (Root Mean Squared Log Error)
* Mean Absolute Error (MAE)
* R² Score
